# TurtleCam Web Server

**TurtleCamWebServer** is a lightweight web server designed to manage and monitor an autonomous turtle camera system. It provides a simple web interface to configure the device and access recorded media directly from the camera.

The system runs on a Raspberry Pi–based TurtleCam device and is accessible through a local hotspot connection, making it ideal for remote or field deployments without internet access.

---

## Features

- View and download captured images and videos  
- Configure and program the TurtleCam device   
- Lightweight and optimized for embedded systems  

---

## Accessing the Web Server

1. Power on the TurtleCam device.
2. Connect your computer or mobile device to the TurtleCam Wi-Fi hotspot.
3. Open a web browser.
4. Navigate to the device’s local IP address.
